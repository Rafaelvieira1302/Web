É possível incluir CSS em documentos HTML de três maneiras principais: inline (usando o atributo style), interno (com a tag "style" no "head") e externo (linkando a um arquivo CSS separado com a tag "link"). O método externo é o mais recomendado para projetos maiores por permitir reutilização e organização, enquanto os métodos interno e inline são úteis para estilos de página única ou ajustes rápidos.
