Formularios

O html possui a tag form que possibilidade definir formularios para submissão de dados informados pelo usuario por meio de campos de impot, dentro do formulario. A tag <form> possui além dos atributos globais, e tambem atributos action="url", utilizado para definir o destino dos dados submetidos. Exemplo: 

        <from action="url">
            <input type="text" name="variavelA"
        </form>

Table

A tag table é utilizada para definição de table no html, internamente em um table podemos criar linhas "rows" utilizando a tag <tr> e celulas dentro das linhas utilizando a tag <td> Exemplo: 

        <table>
            <tr>
                <td>Celula 1x1 </td>
                <td>Celula 1x2</td>
            </tr>
            <tr>
                <td>Celula 2x1 </td> 
                <td>Celula 2x2</td>
            </tr>
        </table>
O codigo anterior define a tabela abaixo
Celula 1x1 Celula 1x2
Celula 2x1 Celula 2x2

Além disso podemos manipular o numero de espaços utilizado em linha e colunas por uma celula utilizado rowspan e colspan
